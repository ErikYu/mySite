基本概念

1. 词典包含两个部分：键(key) 和 值(value)

2. 词典的元素没有顺序，无下标引用，通过键引用。

dic = {'tom':11,'mac':13,'plc':15}

print 'the type of dic is ',type(dic)

print dic

print 'tom scores ',dic['tom']

3. 构建一个空的词典，并添加一个新元素。添加新元素的方法为引用一个新的键并赋值。

eptd = {}

print eptd

eptd['lilei'] = 99

print eptd
